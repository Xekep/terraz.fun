                                <div class="su-spoiler-content su-clearfix" style="display:none">
                                  <strong>Создание региона</strong></p> <ol>
                                    <li>
                                      Наведите курсор мыши на первую точку региона.
                                    </li>
                                    <li>
                                      Нажмите клавишу Q.
                                    </li>
                                    <li>
                                      Не отпуская клавишу Q наведите курсор мыши на вторую точку региона.
                                    </li>
                                    <li>
                                      Отпустите клавишу Q.
                                    </li>
                                    <li>
                                      Наведите курсор мышки на игрока*, нажмите клавишу Q.
                                    </li>
                                    <li>
                                      Уведите в сторону(пустое место) курсор мышки, нажмите Q. Готово.
                                    </li>
                                  </ol>
                                  
                                  <p>
                                    <strong>Проверка наличия региона</strong>
                                  </p>
                                  
                                  <ol>
                                    <li>
                                      Наведите курсор мыши на область, где предположительно есть регион.
                                    </li>
                                    <li>
                                      Нажмите клавишу N (N &#8212; выводит на экран название региона).
                                    </li>
                                  </ol>
                                  
                                  <p>
                                    <strong>Добавление игроков в уже созданный регион</strong>
                                  </p>
                                  
                                  <ol>
                                    <li>
                                      Наведите курсор мыши на область в которой находится регион.
                                    </li>
                                    <li>
                                      Нажмите клавишу N (N &#8212; выводит на экран название региона).
                                    </li>
                                    <li>
                                      Наведите курсор мыши на игрока, которого необходимо добавить в регион, нажмите клавишу Q.
                                    </li>
                                    <li>
                                      Уведите в сторону(пустое место) курсор мыши, нажмите Q.
                                    </li>
                                  </ol>
                                  
                                  <p>
                                    <strong>Удаление региона</strong>
                                  </p>
                                  
                                  <ol>
                                    <li>
                                      Наведите курсор мыши на область с регионом.
                                    </li>
                                    <li>
                                      Нажмите клавишу N (N &#8212; выводит на экран название региона).
                                    </li>
                                    <li>
                                      Нажмите сочетание клавиш Ctrl+X для удаления региона.
                                    </li>
                                  </ol>
                                  
                                  <p>
                                    <strong>Добавление себя в регион</strong>
                                  </p>
                                  
                                  <ol>
                                    <li>
                                      После выделения области на Q, нажмите F6 для добавления себя в регион
                                    </li>
                                  </ol>
